Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

  • HTML is simple to use and understand
  • Programmers can make web pages more interactive by adding images, videos and music.
  • HTML is platform independant and can be used on Windows, Linux and others

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

  • CSS saves time
  • Superior styles to HTML

JavaScript is a high level, dynamic, untyped and interpreted programming language. It has been standardized in the ECMAScrpt language specification. Alongside HTML and CSS, it is one of the 3 essential components of the World Wide Web content production; the majority of websites employ it and it is supported by all morden web browsers without plugins or any king of other extention.
